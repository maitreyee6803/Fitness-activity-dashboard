# Fitness-activity-dashboard
This project is a full-stack fitness tracking system developed using Python, Streamlit, and MySQL. It allows users to log workout activities, store data in a relational database, and visualize fitness progress through interactive charts.
# Techstack:
Python
Streamlit (for UI)
MySQL (database)
Pandas (data processing)
Matplotlib (visualization)
# features:
Add daily workout activities (exercise type, duration, calories)
Store data in MySQL database
View complete workout history
Calculate total calories burned and average duration
Visualize daily calorie trends using graphs
